# Landing

## Quickstart

Install all dependencies for the project:

```sh
yarn
```

Run development server:

```sh
yarn start
```

Create *out* folder:

```sh
yarn run export
```

## Docker

Run development server:

```sh
make run
```

Create *out* folder:

```sh
make export
```
